Game of Life

The post is just some fun with a classic cellurar automaton. The rules are described in detail on Wikipedia with all relevant information. The game below is the final version which allows you to set the initial state and then observe the evolution.

Rules

The rules are copied from Wikipedia with some additional clarification.

  • The game is played on infinite plane. In practice this would mean that objects are moving from one end of plane to other or that we don't track the objects that left the plane.
  • Each cell has 8 neighbours and can be in 2 states. Live or dead.
  • Any live cell with fewer than two live neighbours dies, as if caused by underpopulation.
  • Any live cell with two or three live neighbours lives on to the next generation.
  • Any live cell with more than three live neighbours dies, as if by overpopulation.
  • Any dead cell with exactly three live neighbours becomes a live cell, as if by reproduction.

I will use D3 to animate the changes over SVG component. The game itself is a simple class which allows to iterate and view the evolution of life based on the rules above.
